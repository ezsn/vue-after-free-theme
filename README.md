# Vue After Free Theme (v1.4)

Theme pack for **Vue After Free v1.4**.

## Install

1. **Open Vue After Free on your PS4**  
   > Important: Vue After Free must be running while you use FTP.

2. Connect to your PS4 using your FTP client.

3. Upload/copy this theme folder to:
   `/mnt/sandbox/CUSA00960_000/download0/themes/`

## Enable the Theme

1. Open **Config** inside Vue After Free.
2. Go to **Theme**.
3. Select your theme name from the list.

## Notes

- If the `/mnt/sandbox/CUSA00960_000/download0/themes/` folder does not exist, run Vue After Free once, then try again via FTP.
- Folder name is case-sensitive on some setups. Keep the theme folder name exactly as provided.
